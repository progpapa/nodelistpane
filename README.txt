With this module you can create panel panes to easily display ordered lists of nodes.

Usage:

- Add content to a region in one of your panels.
- You'll see a "Custom Panes" section in the pop-up window, select "Node List".
- You can set the number of nodes to display and choose a view mode. The module defines a custom view mode which is the default selection.
- On the next page you can select which  nodes to display, you can easily re-order the nodes using drag'n'drop.
